# LCDman
 A small game that can be played on the LCD screen
<h1> Components and supplies </h1>
<ul>
 <li>Arduino Uno</li>
 <li>2x16 LCD Screen</li>
 <li>Breadboard</li>
 <li>Resistor (220 ohm)</li>
 <li>Button</li>
 <li>Jumper Wires</li>
 <li>USB Cable for Arduino</li>
</ul>
